# homebrew-agentmail

Homebrew tap for AgentMail - Inter-agent communication for tmux sessions.

## Installation

```bash
brew install UserAd/agentmail/agentmail
```

Or add the tap first:

```bash
brew tap UserAd/agentmail
brew install agentmail
```

## What is AgentMail?

AgentMail is a CLI tool that enables inter-agent communication within tmux sessions. Agents running in different tmux windows can send and receive messages through a simple mail system.

For more information, see the [main repository](https://github.com/UserAd/AgentMail).

## Requirements

- macOS (Intel or Apple Silicon) or Linux (amd64)
- tmux (for full functionality)
